Colophon

Written By: Zachary Kai » Published: 10 Sep 2025 | Updated: 16 Oct 2025

Here, you'll find my colophon: all information pertaining to how this site runs.

This site runs on an Ubuntu Virtual Private Server hosted by Vultr. It uses Static HTML, CSS for styling, and a few php scripts for tools and automations. I use VSCodium to write in, and SFTP to transfer files.

I use the Pirata One font for headings, and Vollkhorn for the body text.

Many thanks to James for the inspiration on the sidebar layout, and Robin Sloan for the typography.

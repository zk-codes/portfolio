How It's Built

This site runs on an Ubuntu Virtual Private Server hosted by Vultr.

It uses static HTML, CSS for styling, and a few PHP scripts for tools and automations. I use VSCodium to write in, and SFTP to transfer files.

Typography

I use Pirata One for headings and Vollkorn for body text. Both are free, open-source fonts served locally.
